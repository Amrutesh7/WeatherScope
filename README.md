# WeatherScope 🌦️

WeatherScope is a full-stack weather application that allows users to search for real-time weather conditions of any city.

The application demonstrates how a frontend can communicate with a backend API to fetch and display data from external services.

---

## Features

- Search weather by city
- Real-time weather data
- Clean and responsive UI
- Full-stack architecture
- API integration using FastAPI

---

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python
- FastAPI

External APIs:
- Open-Meteo Geocoding API
- Open-Meteo Weather API
